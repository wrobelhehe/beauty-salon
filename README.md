# Beauty Salon Landing — Serene Atelier

This repository contains a luxury beauty salon landing page built with Next.js (App Router) and Tailwind CSS, aligned with the creative blueprint in `docs/landing-page-concept.md`. The experience blends cinematic GSAP motion, layered gradients, and tactile UI components.

## Getting started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Lint the project: `npm run lint`

## Structure
- `app/` — App Router entry, layout, and hero-to-footer sections.
- `app/components/` — Reusable UI primitives and animation wrappers.
- `app/lib/` — Content data for services, testimonials, and gallery.
- `docs/landing-page-concept.md` — Creative direction and implementation blueprint.
